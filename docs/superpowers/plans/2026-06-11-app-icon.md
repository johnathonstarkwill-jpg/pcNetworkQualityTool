# App Icon Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the default Electron icon with the custom V2 brand icon (Wi‑Fi signal arcs + green verified badge on a blue rounded square) for packaged macOS and Windows builds.

**Architecture:** A committed SVG source at `build/icon.svg` is rasterized to `build/icon.png` (1024×1024) by a Playwright script; electron-builder generates the per-platform `.icns`/`.ico` from that PNG at package time; the dev window points at the PNG (guarded).

**Tech Stack:** Electron, electron-builder, Playwright (rendering), TypeScript.

- [ ] **Step 2: Write `scripts/render-icon.mjs`**

```js
// Render build/icon.svg -> build/icon.png (1024x1024) with transparent corners.
// Deterministic, headless. Run via `npm run icon`.
import { chromium } from "playwright-core";
import { readFileSync, existsSync } from "node:fs";
import path from "node:path";

const ROOT = path.resolve(import.meta.dirname, "..");
const SVG = path.join(ROOT, "build", "icon.svg");
const PNG = path.join(ROOT, "build", "icon.png");

if (!existsSync(SVG)) {
  console.error(`Missing ${SVG}`);
  process.exit(1);
}

const svg = readFileSync(SVG, "utf8");
const browser = await chromium.launch();
try {
  const page = await browser.newPage({ viewport: { width: 1024, height: 1024 }, deviceScaleFactor: 1 });
  await page.setContent(`<body style="margin:0">${svg}</body>`);
  await page.waitForTimeout(200);
  await page.screenshot({ path: PNG, omitBackground: true });
  console.log(`Wrote ${PNG}`);
} finally {
  await browser.close();
}
```

## Task 2: Wire the icon into packaging + dev window

**Files:**
- Modify: `electron-builder.yml`
- Modify: `src/main/main.ts`

- [ ] **Step 1: Point electron-builder at the icon**

In `electron-builder.yml`, replace the `mac:` and `win:` blocks:

```yaml
mac:
  target:
    - dmg
win:
  target:
    - nsis
```

with:

```yaml
mac:
  icon: build/icon.png
  target:
    - dmg
win:
  icon: build/icon.png
  target:
    - nsis
```

- [ ] **Step 2: Set the dev-window icon (guarded)**

In `src/main/main.ts`:

(a) Add an import for `existsSync` near the other node imports:

```ts
import { existsSync } from "node:fs";
```

(b) Inside `createWindow`, before `mainWindow = new BrowserWindow({`, resolve the icon path:

```ts
  const iconPath = path.join(__dirname, "../../build/icon.png");
```

(c) Add the `icon` option to the `BrowserWindow` options object (top level, NOT inside `webPreferences`), guarded so a missing file never breaks window creation. Change:

```ts
  mainWindow = new BrowserWindow({
    width: 1200,
    height: 800,
    minWidth: 960,
    minHeight: 640,
    title: "PC Network Quality Tool",
    webPreferences: {
```

to:

```ts
  mainWindow = new BrowserWindow({
    width: 1200,
    height: 800,
    minWidth: 960,
    minHeight: 640,
    title: "PC Network Quality Tool",
    ...(existsSync(iconPath) ? { icon: iconPath } : {}),
    webPreferences: {
```

Expected: clean.

Run: `npm test`
Expected: 49 pass (unaffected).
